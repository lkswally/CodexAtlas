# Codex-Atlas Roadmap

## V3

Estado: RC1

Codex-Atlas V3 is frozen as the RC1 baseline. It accepts only:

- P0 fixes.
- P1 fixes.

No new features, registries, policies, workflows, readiness surfaces, runtime
systems, or external integrations should be added to V3.

V3 baseline includes:

- Governance and `atlas_verify`.
- Evidence Pipeline V1.
- Atlas health report V1.
- Workflow Observations Cache with freshness policy.
- Failure Registry V1.
- Model Routing V1 advisory mode.
- Manual Agent Loop design.
- CI workflows and release documentation.

## V4

Preliminary objectives only:

- Autonomous Runtime.
- Runtime Planner.
- Runtime Verifier.
- Runtime Failure Learning.
- Automatic Model Routing.
- Live Workflow Observations.
- Runtime Memory.
- Multi-model orchestration.
- Optional Paperclip integration.
- Optional Engram integration.
- Runtime health interface.

Do not design or implement V4 in the V3 branch. V4 begins only after an
explicit planning phase.

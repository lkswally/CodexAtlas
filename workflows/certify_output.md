# Workflow: certify_output

## Purpose

Define a final review gate before Atlas marks an output as ready for handoff.

## Checks

1. The scope matches the request
2. Claims are supported by repo evidence
3. Boundaries between Atlas and derived projects remain intact
4. Validation is proportional to the risk
5. Residual risks and rollback are explicit

## Output

- certification note
- validation summary
- residual risks

## Status

Documentary only in Level 1.
